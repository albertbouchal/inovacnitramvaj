# inovacnitramvaj
## Tracking of Innovate Prague tram

Just click CHECK if you want to check what line the 9277 car is currently running (this tram currently as of March 2026 has the Innovate Prague livery).
It should work for any other tram "identifikační číslo vozidla" number.

When toggling "Monitor", it refreshes any x seconds, leaving 30 as it is easy on the backend and API requests.

Made with Claude Opus 4.6, free version
