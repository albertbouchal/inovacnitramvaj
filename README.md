# inovacnitramvaj
## Tracking of Innovate Prague tram

Just click TRACK if you want to track the 9277 car (that currently as of March 2026 has the Innovate Prague livery).
It should work for any other tram "identifikační číslo vozidla" number.

It refreshes any x seconds, leaving 20 as it is easy on the backend and API requests.

Made with Claude Opus 4.6, free version
